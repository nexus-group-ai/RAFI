How to Use RAFI

  1. Upload File: Choose a CSV or Excel file containing participant data
  2. Required Columns: Ensure your file has "First Name", "Last Name", and "Attendance" columns
  3. Attendance Filter: Only participants with attendance = 1 will be eligible
  4. Pick Winner: Click the "Pick a Winner!" button to start the selection
  5. Celebrate: Watch the animation and see the winner!

File Format Requirements

Required Columns:

  • First Name or first name
  • Last Name or last name
  • Attendance or attendance

Attendance Values:

  • 1 = Present (eligible for raffle)
  • 0 = Absent (not eligible)

Supported Formats: CSV (.csv), Excel (.xlsx, .xls)

About RAFI

RAFI (Randomised Automated Fairness Initiative) is a free tool designed to conduct fair and transparent raffles or winner selections from uploaded participant data.

Perfect for events, competitions, giveaways, and any situation where you need to select winners randomly from a list of eligible participants.

Developed by Nexus Group AI to ensure fairness and transparency in winner selection processes.
